# Authentication service

A HTTP server (back end only) with DB support & auth features 

## Main Features
- es6
- express
- mongoose
- passport
- validator
- signin / signup
- token and refresh-tokens
- optional roles by environment variables

## Dependencies
- Node.js
- npm OR yarn
- MongoDB

## Install
```sh
$ npm install
```

## Launch
```sh
$ npm run dev
```
